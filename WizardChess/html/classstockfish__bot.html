Detailed Description

Chess board's API to the stockfish chess engine.

Author
Rustam Mamedov

The chess board requires access to a chess engine in order to make decisions on the next move. The class provides an API between the chess board and the stockfish binary, with bidirectional communication. The communication is established using the Unix fork and pipe.

bool stockfish_bot::start ( )

Starts the bot.

Communication is performed through Unix pipes. Processor is forked to execute the stockfish binary. First pipe used to send data from chess board to the bot. Second pipe used to send data from bot to chess board.
